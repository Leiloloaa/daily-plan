### 学习笔记

**算法题复习**

针对字符串对称匹配使用栈处理

1、'aavsdfb' => 'vsdfb' 对对碰 删除字符串中的所有相邻的重复项

```js
// 解题思路
// 1、定义一个 stack
// 2、如果 stack 有值并且 循环项 等于 stack pop 出的这项 就结束本次循环 不保存
// 3、否则就保存
const removeDuplicates=function (s) {
  let stack =[]
  for (let i of s) {
    let c = null
    if(stack.length && i === (c = stack.pop())) continue
    c && stack.push(c)
    stack.push(i)
  }
  return stack.split('')
}

```

2、判断括号的有效性 一定要按照正确的合并顺序
`[{}]` => true `[}}]` => false

```js
// 解题思路
// 但遇到对称匹配的问题可以使用栈 栈是先进后出 正好是对称的
// 1、对称问题先判断奇偶性
// 2、遇到左括号就将右括号存到 stack 中
// 3、如果是又括号就判断 stack.pop 弹出的是否相等
let isValid = function (s) {
  // 按位与等于 1 就是奇数
  if((s&1) === 1) return false
  let stack = []
  for (let i of s) {
    let c = s[i]
    switch (c) {
      case '(':
        stack.push(')')
        break;
      case '[':
        stack.push(']')
        break; 
      case '{':
        stack.push('}')
        break;
      default:
        if(c !== stack.pop()){
          return false
        }
        break;
    }
  }
  return stack.length === 0
} 
```

解题思路：处理字符串，我们很多时候都需要遍历。所以根据题目，我们可以尝试着在 for 循环中改变一下

- i 的值为 每隔 2k 个 即 i += 2*k
- 然后在循环内 再使用 while 替换前 k 个字符
- 左指针 l = i-1; 右指针 r = i + k < len ? len : i + k
- while(--l < ++r){解构交换}

css 选择器的优先级，不同的选择器的权重不一样

- !important 第一优先级
- id 选择器 权重 0100
- style 行样式 权重 0100

**LeetCode 每日一题**

左旋转字符串 例如 abcde 输入一个 k =2  => cdeab

- 可以直接从 k 这里一刀切
- 选择将 k 前面的添加到后面 或者是和后面的添加的前面
- 因为我们可以使用 slice(0,开始的length)
- 所以使用后面的添加到前面会比较好

```js
function reverseLeftWord(s,k){
    const len = s.length
    let i = 0
    while(i < len - k){
      s = s[len-1]+s
      i++
    }
    return s.slice(0,len)
}
```
